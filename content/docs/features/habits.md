## Overview

Habits are the daily practices that compound over time to create lasting change. Exponential's habit tracking helps you build consistency, maintain streaks, and connect habits to your larger goals.

### Key Capabilities

- **Flexible frequencies**: Daily, weekly, or custom schedules
- **Streak tracking**: Current and longest streaks
- **Goal linking**: Connect habits to strategic goals
- **Visual calendar**: 90-day activity view
- **Completion details**: Optional notes, duration, and ratings

## Creating Habits

### Basic Setup

1. Navigate to **Habits** from the sidebar
2. Click **New Habit**
3. Enter habit details:
   - **Title**: Clear, action-oriented name
   - **Description**: Context and motivation
   - **Frequency**: How often to complete
   - **Goal**: Linked strategic goal (optional)
4. Click **Create**

### Habit Fields

| Field | Purpose |
|-------|---------|
| Title | Action-oriented habit name |
| Description | Context and motivation |
| Frequency | Completion schedule |
| Time of Day | When to perform (optional) |
| Goal | Strategic goal this supports |

## Frequency Options

Set how often you want to complete each habit:

| Frequency | Schedule |
|-----------|----------|
| Daily | Every day |
| 3x Week | Three times per week |
| Weekly | Once per week |
| Bi-Weekly | Every two weeks |
| Monthly | Once per month |
| Custom | Specific days of week |

### Custom Days

For custom frequency, select specific days:
- 0 = Sunday
- 1 = Monday
- 2 = Tuesday
- 3 = Wednesday
- 4 = Thursday
- 5 = Friday
- 6 = Saturday

**Example**: Mon/Wed/Fri workout = Days 1, 3, 5

## Tracking Habits

### Today's Habits

View habits due today:

1. Go to **Habits** page
2. View "Today's Habits" section
3. Habits due based on frequency and day
4. Check off as completed

### Completing a Habit

Click the checkbox to mark complete. Optionally add:

- **Notes**: What you did, how it went
- **Duration**: Minutes spent
- **Rating**: 1-5 quality/satisfaction score

### Habit Card

Each habit displays:
- Title and description
- Frequency badge
- Current streak count
- Longest streak achieved
- Linked goal (if any)
- 90-day activity calendar

## Streak Tracking

### Current Streak

Consecutive completions counting back from today:
- Daily habits: Consecutive days
- Other frequencies: Recent completions in last 7 days

### Longest Streak

Your all-time best streak for this habit:
- Historical record
- Goal to beat
- Motivation metric

### Streak Breaks

What breaks a streak:
- Missing a required completion day
- For daily: Missing any day
- For 3x week: Fewer than 3 in a week

## Visual Calendar

The streak calendar shows 90 days of activity:

### Calendar View

- Color-coded completion status
- Month/week organization
- Quick visual of consistency
- Pattern recognition

### Reading the Calendar

| Color | Meaning |
|-------|---------|
| Filled | Completed |

## Managing Habits

### Editing Habits

1. Click on habit card
2. Select **Edit**
3. Modify fields
4. Save changes

### Deactivating Habits

Pause without deleting:
1. Edit habit
2. Set to inactive
3. Habit hidden from daily view
4. History preserved

### Deleting Habits

Remove completely:
1. Click on habit card
2. Select **Delete**
3. Confirm deletion
4. All history removed

## Best Practices

### Starting New Habits

- Start small (2 minutes)
- Stack on existing habits
- Make it obvious
- Remove friction

### Maintaining Habits

- Never miss twice in a row
- Track visually (calendar)
- Celebrate small wins
- Review weekly

### Choosing Habits

- Align with goals
- Start with 1-3 habits
- Build momentum before adding
- Focus on keystone habits

### Frequency Selection

- Daily for critical habits
- 3x week for demanding habits
- Custom for specific schedules
- Don't overcommit

### Goal Connection

- Link every habit to a goal
- Understand the "why"
- Review alignment regularly
- Adjust as goals change

### Recovery from Breaks

- Don't abandon after a miss
- Resume immediately
- Analyze what caused the break
- Adjust if needed
- Rebuild streak
