## Overview

The Wheel of Life is a holistic life assessment tool that helps you evaluate balance across key life domains. By regularly assessing your priorities and satisfaction, you can identify where to focus your energy for maximum life improvement.

### Key Capabilities

- **10 life domains**: Comprehensive life coverage
- **Priority ranking**: Understand what matters most
- **Satisfaction scoring**: Track current fulfillment
- **Gap analysis**: Identify underinvested areas
- **AI recommendations**: Get improvement suggestions
- **Quarterly tracking**: Monitor progress over time

## Life Domains

The assessment covers 10 core life domains:

| Domain | Description |
|--------|-------------|
| Career/Business | Professional life, job satisfaction, career growth |
| Finance/Wealth | Financial security, savings, investments |
| Health/Fitness | Physical health, exercise, nutrition, sleep |
| Family/Relationships | Relationships with family members |
| Romance/Partnership | Romantic relationships and intimacy |
| Personal Growth | Learning, self-improvement, skills |
| Fun/Recreation | Hobbies, leisure, enjoyment, play |
| Physical Environment | Home, workspace, surroundings |
| Social/Friends | Friendships, social connections, community |
| Spirituality/Purpose | Meaning, purpose, values, spiritual practices |

## Assessment Modes

### Quick Mode

A fast assessment focusing on priorities only:

1. **Current Priority**: Rank domains 1-10 (1 = highest priority)
2. **Desired Priority**: How you want priorities to be
3. **Optional Notes**: Reflections on each domain

Best for:
- Regular check-ins
- Time-constrained sessions
- Priority clarification

### Deep Mode

Comprehensive assessment with satisfaction scoring:

1. **Current Priority**: Rank domains 1-10
2. **Desired Priority**: Desired priority ranking
3. **Satisfaction Score**: Rate each domain 1-10
4. **Reflections**: Detailed notes per domain

Best for:
- Quarterly assessments
- Major life transitions
- Detailed planning

## Taking an Assessment

### Starting

1. Navigate to **Wheel of Life** from the sidebar
2. Click **New Assessment**
3. Select **Quick** or **Deep** mode
4. Begin the assessment wizard

### Priority Ranking (Step 1)

Drag and drop domains to rank current priorities:
- Position 1 = Highest priority
- Position 10 = Lowest priority
- Be honest about current state

### Desired Priority (Step 2)

Rank how you want priorities to be:
- Consider ideal life balance
- Think long-term
- Don't overthink—go with instinct

### Satisfaction Scoring (Deep Mode Only)

Rate each domain 1-10:
- 1-3: Very unsatisfied
- 4-5: Below average
- 6-7: Acceptable
- 8-9: Good
- 10: Excellent

### Completion

- Add final notes
- Review your rankings
- Click **Complete Assessment**

## Understanding Results

### Priority Gap Analysis

The gap between current and desired priority reveals investment needs:

| Gap Type | Meaning | Action |
|----------|---------|--------|
| Positive Gap | Underinvesting | Increase focus |
| Zero Gap | Balanced | Maintain |
| Negative Gap | Overinvesting | Consider reducing |

**Example**:
- Current: Career = #1
- Desired: Career = #4
- Gap = -3 (overinvesting in career)

### Attention Indicators

Domains flagged for attention when:
- Positive priority gap (underinvesting)
- Satisfaction score ≤ 5 (Deep mode)

### Dashboard View

After assessment, view:
- Latest assessment summary
- Priority gap chart
- Domains needing attention
- Historical comparison
- AI recommendations

## Best Practices

### Assessment Frequency

- **Quarterly**: Full assessment (recommended)
- **Monthly**: Quick check-in (optional)
- **Weekly**: Mental review in weekly review

### Honest Assessment

- Rate current state, not aspirational
- Don't judge—just observe
- Include uncomfortable truths
- Trust your initial instincts

### Balanced Action

- Don't try to fix everything at once
- Focus on 1-2 priority gaps
- Make sustainable changes
- Reassess before adding more

### Using Results

- Create goals for priority gaps
- Set outcomes for improvement
- Build habits supporting change
- Track progress quarterly

### Domain-Specific Strategies

**Low-Satisfaction Domains**:
1. Identify specific issues
2. Create measurable goal
3. Set weekly outcomes
4. Build supporting habits

**Overinvested Domains**:
1. Question if sustainable
2. Consider delegation
3. Set boundaries
4. Redirect energy

**Balanced Domains**:
1. Maintain current approach
2. Document what's working
3. Don't neglect
4. Periodic check-ins
