## Welcome to Exponential

Exponential is a productivity management application that helps you organize projects, track actions, and achieve your goals with AI assistance.

## Key Features

- **Project Management** - Create and track projects with status, priority, and progress tracking
- **Action Management** - Task management with flexible priority system
- **Goal & Outcome Tracking** - Hierarchical goal-outcome-action alignment
- **Daily Planning** - Journal system with reflection and planning tools
- **AI Assistant** - Chat interface with intelligent task suggestions
- **Integrations** - Connect with Slack, Fireflies, and more
