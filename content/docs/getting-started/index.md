## Getting Started

This quickstart guide will help you get set up with Exponential and start managing your projects and tasks.

## Create Your Account

Sign up for Exponential using one of the following authentication methods:

- **Google** - Sign in with your Google account
- **Discord** - Sign in with Discord
- **Notion** - Connect with your Notion workspace

## Create Your First Project

Once you're signed in, create your first project:

1. Click the **New Project** button in the sidebar
2. Enter a project name and description
3. Set the project status and priority
4. Click **Create**

## Add Actions

Actions are the tasks within your projects:

1. Navigate to your project
2. Click **Add Action**
3. Enter the action details
4. Set due date and priority as needed
